# 文档与结构索引

## 1. 目录结构
- `backend/`：FastAPI 后端
- `frontend/`：React 前端
- `data/`：原始与处理后的数据
- `docs/`：项目文档
- `scripts/`：数据清洗与验证脚本
- `tests/`：测试脚本

## 2. 主要文档
- `docs/requirements.md`：需求文档
- `docs/design_document.md`：系统设计说明
- `docs/Deployment_Guide.md`：部署指南
- `docs/User_Manual.md`：用户手册
- `docs/2025_integration_changelog.md`：变更记录
- `docs/audit/data_usage_report.md`：数据口径审计

## 3. 数据口径
- 原始数据：`data/data.xlsx`
- 运行时数据：`data/processed/enterprises.csv`

## 4. 清理说明
- 已删除未实现/已实现清单、开题报告、进度计划等冗余文档
- `docs/` 仅保留可维护的核心文档
