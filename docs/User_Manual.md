# 用户手册

## 1. 系统简介
本系统为“湖北省人工智能产业生态大脑”可视化平台，结合真实 CSV 数据与产业报告口径指标，提供多维分析与交互展示。

## 2. 核心功能
- 战略情报 / 战略全景 / 创新生态
- 产业数据看板 / 行业分析 / 产业链图谱
- 企业画像 / 区域-产业矩阵
- 算力基建 / 数据要素看板

## 3. 双模界面切换
- **经典模式（V1）**：适合一线操作，卡片边框清晰、信息密度高
- **体验模式（V2）**：适合汇报展示，留白更大、标题更突出、微质感背景
- 在侧边栏模式切换区一键切换，无需刷新

## 4. 常用操作
### 4.1 搜索与筛选
在“产业数据看板”顶部搜索框输入关键词，或通过区域/产业链筛选，图表与列表自动联动。

### 4.2 导出数据
在“产业数据看板”点击“导出报告”，下载当前筛选结果的 CSV 文件。

### 4.3 企业画像
在企业列表点击企业名称，可跳转至企业画像页面查看详细分析。

## 5. 口径说明
- 宏观 KPI（如 1000 亿、1326 家）为展示口径
- 企业列表与图表统计以 `data/processed/enterprises.csv` 为准
